AGENT 01 🔍

Triage QA

Classifier

Localization

Computes Grad-CAM attention weights to construct a pixel heatmap overlay mapping the exact visual findings.

AGENT 04 🗂️

Reasoning (RAG)

Pulls local semantic WHO and national clinical guidelines corresponding to detected diseases for context validation.

AGENT 05 📝

Reporter

Generates structured clinical reports and patient summaries translated into English and Hindi using LLM generation.
